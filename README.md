# Распознавание конфликтов на видео

Дипломный проект. Разработано приложение для автоматического обнаружения драк в видеозаписях с использованием YOLOv8. Выполнено сравнение моделей, подбор порога, создан графический интерфейс (PyQt5) и настроен запуск через bat-файл.

## 📁 Полная структура репозитория
diplom/
├── fight_detector_app.py # Главное приложение (PyQt5)
├── yolo_small_weights.pt # Финальная модель (small)
├── Yolo_nano_weights.pt # Модель nano для сравнения
├── requirements.txt # Зависимости
├── zapusk diploma.bat # Скрипт для запуска приложения
├── icon.ico # Иконка окна
├── fight.mp4 # Тестовое видео с дракой
├── видео для теста.mp4 # Дополнительное тестовое видео
├── my_videos/ # Папка с датасетом (2000 видео: fight/non_fight)
├── runs/ # Результаты детекции (примеры)
├── diplomchuk sravnenie.ipynb # Ноутбук сравнения small vs nano
├── diplomchuk test.ipynb # Ноутбук тестирования модели
├── obuchenie.ipynb # Эксперименты с дообучением (неудачные)
└── gotovaya rabota.ipynb # Итоговый ноутбук с анализом

## 🚀 Запуск приложения

1. Установите зависимости (Conda рекомендуется):
   ```bash
   conda create -n fight_env python=3.10 -y
   conda activate fight_env
   pip install -r requirements.txt
       Дважды кликните zapusk diploma.bat (или выполните python fight_detector_app.py).
